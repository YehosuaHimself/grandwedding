# Toni & Eva — Wedding Site

Deployed at **[yhgrandwedding.love](https://yhgrandwedding.love)**

## Structure
- `index.html` — full single-page site (login, guest portal, host/atelier portal, planning tool)
- `c*.webp` — venue photographs (Il Castello di Piacenza), watermark-stripped
- `CNAME` — GitHub Pages custom domain configuration

## Theme
"Miracle Wedding" — restrained luxury, gold-on-ivory typography, navy micro-accents, the chapel as sacred heart.

## Login
Built-in roles: guest, client, planner. See `USERS` object in `<script>` for credentials.
